# Enterprise SOC Detection Lab

## Overview

This project demonstrates the implementation of an enterprise Security Operations Center (SOC) detection lab using Splunk Enterprise, Sysmon, Splunk Universal Forwarder, and Atomic Red Team.

## Tools Used

- Splunk Enterprise
- Splunk Universal Forwarder
- Sysmon
- Atomic Red Team
- PowerShell

## Features

- Windows event log collection
- Security monitoring
- Threat detection
- Alert generation
- Dashboard creation
- MITRE ATT&CK mapping

## Attack Simulations

- PowerShell execution
- Encoded PowerShell commands
- User account creation
- Scheduled task creation
- Service creation

## MITRE ATT&CK Techniques

| Technique ID | Description |
|---|---|
| T1059.001 | PowerShell |
| T1136 | Create Account |
| T1053 | Scheduled Task |
| T1543 | Create or Modify System Process |
